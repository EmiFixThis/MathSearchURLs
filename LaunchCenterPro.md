# Math Search URLs

This set is formated for Launch Center Pro, I tested the first one. 


### Open problem garden

http://www.openproblemgarden.org/search/[prompt:URL]

### Wolfram MathWorld

http://mathworld.wolfram.com/search/?query=[prompt:URL]&x=0&y=0

### Search Math Wikia

http://math.wikia.com/wiki/Special:Search?search=[prompt:URL]&fulltext=Search

http://www.artofproblemsolving.com/wiki/index.php?title=Special%3ASearch&search=[prompt:URL]&fulltext=Search
